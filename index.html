2️⃣ 좌측 메뉴 Realtime Database → 데이터베이스 만들기 (테스트 모드)
3️⃣ 프로젝트 설정 ⚙️ → 내 앱 → 웹 앱 추가 → SDK 설정 복사
4️⃣ 이 HTML 파일 맨 아래 <script> 안의 FIREBASE_CONFIG 부분을 복사한 값으로 교체
